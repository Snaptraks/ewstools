
ewstools
--------

A Python package for computing, analysing and visualising early warning signals (EWS) 
in time-series data.

.. toctree::
   :maxdepth: 2
   :caption: Contents

   ewstools
   



Installation
============

Install ewstools with pip

.. code-block:: shell

    pip install ewstools
    
Demos
========
